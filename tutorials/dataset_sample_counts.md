## 数据集说明

### MIX_large 构成数据集（15 个）

这 15 个数据集共同构成 MIX_large。其中 Tongji 在 data loader 中被细分为 Tongji1 / Tongji2 / Tongji3（按电池编号前缀划分），所以 MIX_large 实际覆盖 13 个物理数据集但映射为 15 个训练数据集。

### 独立数据集（9 个）

以下数据集不属于 MIX_large，各自独立训练和评估：

- **ZN-coin / ZN42 / ZN2024**：ZN-coin 数据集的三种 seen/unseen 协议划分变体
- **CALB / CALB42 / CALB2024**：CALB 数据集的三种 seen/unseen 协议划分变体
- **NA-ion / NAion42 / NAion2024**：NA-ion 数据集的三种 seen/unseen 协议划分变体

### 组合数据集（1 个）

- **MIX_large**：由 13 个物理数据集合并而成（HUST + MATR + SNL + RWTH + MICH + MICH_EXP + UL_PUR + CALCE + HNEI + Tongji + Stanford + ISU_ILCC + XJTU）

### 未纳入训练的数据集

- **SDU** 和 **Stanford_2**：这两个数据集存在于 `extracted_features/` 中，但 `data_split_recorder.py` 中未定义其训练/验证/测试集划分，因此未包含在上述 25 个数据集中。
